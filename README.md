# RandomMath
 game to find a random number in java


# DEPENDECIES

1) ``` git clone https://github.com/MiroYld/RandomMath.git```
2) ```sudo add-apt-repository ppa:linuxuprising/java```

3) ```sudo apt update```

4) ```sudo apt install oracle-java17-installer```

5) ```java -version```

# TO EXECUTE

1) ```cd RandomMath/src```

2) ```java RandomMath.java```
